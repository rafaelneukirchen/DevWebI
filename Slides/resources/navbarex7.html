Barra de Navegação Vertical

Neste exemplo centralizamos o texto e criamos bordas externas na <ul> e internar na <li>.

Barra de Navegação Lateral fixa e com altura máxima.

Deslize a barra de rolagem para ver como está lista de links continua fixa a página.

Note que o elemento div tem uma margem de 25%. Isto porque a barra de links tem uma largura definida de 25%. Se removermos a margem, a barra lateral vai sobrepor sobre a div.

Note também que setamos overflow:auto para a barra lateral. Isto vai incluir uma barra de rolagem caso a sidenav seja muito longa.
